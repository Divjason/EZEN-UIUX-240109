- Canvas API => 웹 문서상에 그림을 그리거나 디자인을 작업을 하게 도와준다
- Localstorage API

1.canvas API를 활용해서 웹브라우저에 무언가를 그리겠다라고 한다면, 반드시 <canvas>라는 태그를 활용!!!

2.<canvas>태그를 사용할 때에는 반드시 사이즈에 대한 정의가 필수!!! 하지만, 만약 사이즈 정의가 되지 않았을 시에는 기본값으로 설정 (*300 x 150px)

3.만약 웹 브라우저 전체 화면을 canvas화 하고자 한다면, JS를 활용해서 window객체의 inner를 활용한다

4.canvas태그를 활용해서 웹브라우저 문서상에 실제 캔버스를 만들었다면, 그안에 그림을 그릴 수 있음

5.canvas태그안에 그림을 그리려면, 반드시 getContext("2d")함수를 활용해서 객체를 생성해줘야함

6.생성된 객체에 사용할 수 있는 다양한 속성 & 함수 존재
- fillStyle 속성 : 채우기 컬러를 정의
- fillRect 함수 : 사각형의 도형을 생성

7.곡선이 그려지는 원리!!!
시작점 / 종료지점은 직선과 동일하게 존재!!!
시작점 & 종료지점 사이 어딘가에 조절점이 존재!!!
조절점의 위치에 따라서 곡선이 그려짐!!!
이때 생성되어지는 곡선을 베지에 곡선이라고 함
- 조절점이 하나일 때 생성되어지는 곡선 = 2차 베지에 곡선
- 조절점이 두개일 때 생성되어지는 곡선 = 3차 베지에 곡선

8.이미지를 잘라낸다는 개념
1) 어느 지점에서 부터 이미지를 잘라낼 것인지, 좌표값
2) 해당 좌표값으로부터 얼만큼 떨어진 지점인지
3) 해당 좌표값을 기준으로 w / h 어떻게 되는지
4) 잘라낸 이미지를 어떤 w / h 값으로 출력
