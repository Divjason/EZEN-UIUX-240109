*셀렉토 커피 프랜차이즈 매장 정보 클론코딩!!!

1.맵 특정 지역명 클릭
- select 태그 정보 변경
- 위젯 스피너 로딩
- 특정 지역명 스타일 정의
- upgrade : 도시명 변경 => 구/군 정보 매칭!!
- upgrade : 우측 select 태그 정보에 따라 맵 변경

2.매장 서비스별 옵션 선택
- 해당 옵션 선택, 해당 매장 노출
- input태그 내 특정 매장명 입력 검색버튼 기능!!!

3.마우스 커서 이동 시, 아이콘 이동

4.페이지네이션 기능
- 1개의 페이저 당 5개 매장 노출
- 페이저 5개가 하나의 그룹
- 라스트페이지 바로가기 기능
- 페이저 prev & next 존재
- 선택된 페이지를 정의할 수 있는 스타일가이드

5.(new)특정 버튼 클릭, 마우스 아이콘 한바퀴 회전 기능

1개의 페이지에 5개의 도시 정보를 출력!!
5개의 페이저가 1개의 그룹!!!
1번 그룹의 첫번째 페이저 1번 페이지
2번 그룹의 첫번째 페이저 6번 페이지
