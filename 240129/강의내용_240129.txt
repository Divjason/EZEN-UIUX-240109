2.css3 : 스타일시트

3.js : 프로그래밍 언어 중 하나!!!
=> 웹브라우저 화면 ui 컨트롤하기 위해서 반드시 초급단계에서 공부해야하는 js / java / c / c++ / c# / python / 아두이노 / 바벨 / 웹팩등등
=> 공은 이렇게 차는거야
=> A의 상황에서는 공을 살짝 힘을 줘서 차야지
=> B의 상황에서는 공을 살짝 힘을 빼고 차야지

코드를 작성하기에 앞어서 내가 현재 구현하고자 하는 기능이 무엇인지 부터 정의하고 가는게 좋다!!!

1.컴퓨터에게 우측상단 토글 버튼이 존재한다. 라는 것을 알려줘야 함
- JS => 변수(*바구니) 선언 => 토글 버튼을 담아준다

2.컴퓨터에게 gnb 네비게이션이라는 존재 역시 알려줘야 함
- JS => 변수 선언 => gnb 네비게이션을 담아준다

3.우측상단 토글 버튼을 클릭하지 않은 상태일 때에는 gnb 네비게이션이 나타나면 안된다.
- CSS => 특정요소의 존재 자체를 없앨 수 있는 스타일 속성
(*display : none)

4.우측상단 토글 버튼을 클릭했을 때, gnb 네비게이션이 나타난다.
- JS => 이벤트 (User가 어떤 액션 행동을 취했을 때) => 행위를 만들어라 (이벤트 => 리액션을 만들어라!!!)
addEventListner라는 함수(*어떤 기능을 정의해놓은 역할)를 사용할 수 있음
